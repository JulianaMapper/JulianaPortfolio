The situation

Consultants had built the model. The business had adopted it. And it was driving multi-year, multi-million dollar e-commerce facility network decisions — where to build warehouses, which stores to use for fulfillment. Variables were hard-coded. File paths were hard-coded. The model hadn't been questioned.

I questioned it because that's what I do. I don't trust a dataset until I understand its limitations. When I evaluated the Esri consumer spending data that anchored the model, I started asking: does this variable actually predict e-commerce demand, or does it just look like it does? The answer wasn't clean. That's when I knew the real problem wasn't the code — it was that nobody had ever run a validation.

What I heard

Before touching anything, I conducted user assessments and led focus groups to understand how the model was actually being used versus how it was supposed to be used. The gap was significant. People trusted outputs they didn't fully understand because there was nothing better.

The dashboard situation was the same story: ad-hoc requests, constant stress, no repeatable process. I ran interviews to understand what decisions people were actually trying to make.

What I built
Cleaned up the codebase: relative paths, proper date handling, updated model logic
Brought in new spatial data sources as better predictors of e-commerce demand than the originals
Ran PCA — the first validation the model had ever received — to build stakeholder confidence and expand adoption
Systematic dashboard process: SQL → Alteryx → Tableau, cut reporting time from 8 hours to 30 minutes
Set investment priorities for multi-year e-commerce facility network decisions
What changed

Stakeholders who had quietly doubted the outputs now had documented evidence they could trust. The ad-hoc dashboard chaos became a repeatable, user-centric process that the team actually wanted to use.

The real change: multi-million dollar facility decisions started being made on a model that had actually been validated. That's not a small thing.
