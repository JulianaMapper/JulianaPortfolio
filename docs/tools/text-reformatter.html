What this does

Converts markdown (as returned by ChatGPT, Claude, Gemini, etc.) into styled HTML that pastes cleanly into Microsoft Word. Headings map to Word's built-in heading styles. Tables retain alternating row colors. No formatting lost in translation.

Privacy

Entirely client-side. Nothing leaves your browser. No server, no tracking, no data stored. Your content stays yours.

Developer Notes
Show

Preview vs. Word output

The preview uses Still Water colors. When you paste into Word, content adopts your document's Heading 1–6 styles. Table inline styles (row colors) carry over regardless of Word theme.

Format vs. Smart Cleanup

  • Format — minimal processing: bullet normalization, line break cleanup, markdown parse. Preserves your text structure.
  • Smart Cleanup — aggressive: removes LLM preamble phrases, converts all-caps headers, detects table formats (pipe, TSV, space-aligned), normalizes typography.

Technical stack

  • Uses marked.js for markdown → HTML conversion
  • Word-compatible copy via ClipboardItem API with text/html MIME type
  • Table rows get inline background styles at copy time (Word ignores CSS nth-child)
  • Tabler Icons for UI; DM Sans + Outfit for fonts
